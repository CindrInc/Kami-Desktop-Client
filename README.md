## Development
To get started just pull the repo and run the following

```bash
npm install
npm run build
npm start
```

To build the installers / release packages you need to run, you can only build a platforms installer from that platform.

```bash
# Windows
npm run package:win

# Mac OS X
npm run package:mac

# Linux
npm run package:linux
```
